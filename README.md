## 🛠️ Key Learning Areas

### 📊 QuickBooks
- Cloud-based bookkeeping and accounting workflows  
- Invoicing, expenses, bank reconciliation  
- Financial reporting (P&L, Balance Sheet, Trial Balance)  
- Transaction lifecycle in SME accounting  

### 📊 Xero
- Cloud accounting system structure  
- Chart of accounts and financial modules  
- Bank feeds and reconciliation  
- Real-time reporting and collaboration features  
- Comparison with QuickBooks from control perspective  

### ⛓️ Blockchain Basics
- Distributed ledger technology (DLT)  
- Blocks, nodes, consensus mechanisms  
- Transparency and immutability concepts  
- Smart contracts and automation logic  
- Public vs private blockchain systems  

---

## 🔍 Key Insights

- Cloud accounting improves speed, visibility, and control in financial reporting  
- QuickBooks and Xero follow similar accounting logic but differ in workflow and usability  
- Blockchain introduces a **trustless, decentralized model** for recording transactions  
- Smart contracts shift control logic from systems to code-based execution  
- Emerging technologies reshape audit trails, controls, and financial transparency
